# Smart-traffic-monitoring-system-
The Smart Traffic Monitoring System is an intelligent traffic management solution that uses computer vision, IoT, and edge computing to monitor road traffic in real time. The system captures live video from cameras connected to a Raspberry Pi and processes the video using AI-based object detection models such as YOLO (You Only Look Once).

The system detects vehicles, counts traffic density, and identifies emergency vehicles like ambulances, fire trucks, and police vehicles. When an emergency vehicle is detected, the system can prioritize traffic signals to clear the route.

# Main Features
1.Real-time traffic video monitoring
2.Vehicle detection and counting
3.Emergency vehicle recognition
4.Traffic density analysis
5.Smart signal control
# Applications
1.Smart cities
2.Traffic congestion reduction
3.Emergency vehicle priority
4.Urban traffic management
